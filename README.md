# assembler_in_python
an assembler in python which handles add, sub, and, or, xor, test, cmp, mov instructions 
and eax, ebx, ecx, edx registers and their sub registers
and zero Flag, sign Flag, carry Flag, overflow Flag, parity Flag
and it can read binary, decimal and hexadecimal numbers.
it is free-form (the positioning of characters on the programming page is insignificant).
it is not case sensitive.
